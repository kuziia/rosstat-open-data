# Визуализация открытых данных Росстата

Проект «Визуализация открытых данных Росстата» собирает, конвертирует и визуализирует публикуемую [Росстатом](https://rosstat.gov.ru) статистику.  
Данные загружаются из официальных xls‑файлов, преобразуются в SQL‑запросы и подключаются к дашбордам Metabase.

- **Дашборды:** https://metabase.findatalab.ru/  
- **Панель базы данных (pgAdmin):** https://pga2.findatalab.ru/  
- **Исходные данные:** https://rosstat.gov.ru/folder/10705

## Как внести вклад в проект 

0. Сделайте форк и клонируйте репозиторий. 
1. Скачайте нужный xls‑файл со страницы Росстата.
2. Сохраните его в соответствующей тематической папке `data/...`.
3. Для каждой вкладки сформируйте промпт (описание структуры таблицы и данные) в `*.txt` 
4. Конвертируйте таблицу в SQL‑запрос с помощью вашего любимого ИИ помощника (Алиса, Дипсик, Чат GPT и т.д.).
5. Поместите SQL‑запрос в файл `*.sql` и выполните его в pgAdmin или через командную строку. 
6. Создайте дашборд в Metabase, используя новые таблицы. 
7. Если все получилось, можно делать коммит. В коммите должны быть файлы:
    - **`*.xls`** – оригинальный файл, скачанный с сайта Росстата.
    - **`*.txt`** – промпт (инструкция) для генерации SQL‑запроса из таблицы (несколько, по количеству вкладок).
    - **`*.sql`** – готовые SQL‑запросы для создания и заполнения таблиц в PostgreSQL.

8. Сделайте push в свой репозиторий. 
9. Сделайте pull request в основной репозиторий. Прикрепите ссылку на дашборд в комментарии к pull request.
